# Screenshots

Add your application screenshots here:

1. `safe-widget.png` - Main widget screenshot (for README header)
2. `light-mode.png` - Light theme screenshot
3. `dark-mode.png` - Dark theme screenshot
4. `lock-action.png` - Showing the lock/encrypt action
5. `unlock-action.png` - Showing the unlock/decrypt action

## Recommended Screenshot Sizes

- Widget screenshots: Capture at actual size (420x180)
- You can also create 2x resolution versions for retina displays

## How to Take Screenshots

1. Run the app: `npm run tauri dev`
2. Use Windows Snipping Tool (Win + Shift + S)
3. Save as PNG with descriptive names
